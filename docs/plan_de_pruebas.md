## 5) Pruebas mínimas por lenguaje
- **Python**: ejecución de `main.py` con caso normal y caso inválido.
- **C**: compilación y ejecución con caso normal y caso inválido.
- **Bash**: ejecución de script con parámetros válidos e inválidos.
- **ARM64 Assembly**: ensamblado/ejecución mínima con al menos un caso correcto y uno de error controlado.

## 6) Criterios para considerar la práctica terminada

- La documentación coincide con lo implementado.
- El alcance se mantiene pequeño y viable.
